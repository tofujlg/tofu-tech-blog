## 状況
React Nativeでブラウザのインスペクト機能のように、UIの情報を取得する方法。
デバイスを振るとメニューが表示されるというが、なぜか僕の環境ではそれができなかった。以下のコマンドをターミナルで実行すると、デバイスを振らずにメニューを表示できる。

```shell
adb shell input keyevent 82
```

もっと早く知りたかったコマンドではある。`abd`は色々な機能があって便利だ。
