## Main
2023/11 ~ 2024/1の身体運動系について振り返りたいと思う。

## ウェイトトレーニング
まずは平日はほぼ毎日やっているウェイトについて、10月末にベンチプレス100kgを達成したのと、有料のジャンプトレーニングのメニューを3ヶ月ほどこなしていたのが終わったので、11月からは新しいスタートだった。

僕のウェイトは上半身と下半身の日に分かれていて、それぞれ週2回、合計4回やっている。

### 上半身の日

ここ3ヶ月の典型的な上半身のメニューがこちら

| 種目      | セット数 | レップ数 |Max重量 |
| ------- | ---- | ---- | ---- |
| ベンチプレス  | 5-7  | 3RM  |95kg |
| プッシュプレス | 5-7  | 3RM  |65kg |
| 懸垂      | 3    | 6rep    |8rep |


(体重 81kg)

11月末に2週間ほどトレーニングを休んで、体重が5kgほど落ちたので、MAXの重量も落ちた。

特にベンチプレスはこの3ヶ月1回も100kgを上げることができなかった。10月に上げた時は「100kgを絶対に挙げる」という断固たる決意のもと、ベンチだけで50分ぐらいかけるほどセットをこなしていた。達成してから3セットに落としたので、そりゃ弱くなって当然やなぁと思い。最近またセットを増やしている。

### プッシュプレス

プッシュは身体の連動性を高めてくれるし、肩のトレーニングに良いと感じる。ここ2年ぐらいはオーバーヘッドプレスをやっていたが、プッシュに変えてからバスケのパフォーマンスに良い影響を感じる。ジャンプシュートの際に、垂直方向への力の流れがスムーズになっていると感じる。
この種目は3repを3セットとかで、ベンチとボリュームを逆にした方が良いかもしれないとも感じているが、ベンチ100kgを安定的に上げれるようになってからにしようと思う。

### 懸垂
なんか義務感でやっている。体の背面を鍛えてくれる種目でこれより優れているものを知らないから。


### 下半身の日

こっちが本命なのだが、最近また腰が怖くてあまり本気でトレーニングできていない。

典型的なメニューがこちら

| 種目        | セット数 | レップ数 |Max重量 |
| --------- | ---- | ---- | ---- |
| パワークリーン   | 5-7  | 3RM  |80kg |
| フロントスクワット | 5-7  | 3RM  |測定してない(6rep maxは90kg) |
| 垂直跳び      | 3    | -    |70cmぐらい |

とにかくダンクがしたくて色々試行錯誤している。ダンクについては長くなるのでまた別の記事で。

ダンクのためにウェイトで何が一番良いかと、1年ぐらい試行錯誤した結果、僕の場合クリーンの重量とジャンプ力に相関性が高いということに気がついた。クリーンの重量が上がるに連れて、リングから出る指の高さが上がってく。

だから、ウェイトルームでは、只管パワークリーンを伸ばすことに注力している。

### パワークリーン
これも3レップで組んでいる。まだ初めて半年ぐらいしか経っていない種目だが、早速初心者ボーナスステージは終わった感じがする。80kgを超えて、ここからはベンチみたいに地道なトレーニングが必要になってくる。

今の課題としては、

1. キャッチが下手くそということ
	- しゃがみこみの速度が遅い
	- 前傾しているので重さを支えきれない
2. 腕を使いすぎ
	- 股関節のパワーを使えていない

というものがあると思う。

1に対しては、フロントスクワットをすることで強化していこうと思っている。それからまだ始めていないが、ハングクリーンも取り入れていこうと思う。

2に関しては、意識の問題なのかな。
今はどうしたら良いのかわからないので、兎に角足の力をできるだけ使うように、ジャンプするようにバーを持ち上げるようにしている。これに対してもハングクリーンが有効なのでは無いかと思う。

## 垂直跳び
ジムでのジャンプ系はこれしかできない。たまにスペースが空いていたら、踏切フォームの確認とかもしている。そのおかげか、最近ゴールしたからジャンプしてリングに届くようになった。しゃがみ込み速度とジャンプ力の相関性はすごく感じる。

そういえばクリーンアンドジャークを試しにやってみたら70kgできた。

ウェイトに関してはこんな感じ

## バスケ
ほぼ週１、たまに週2

試合もなかったので、普通に練習に参加していた。兎に角走ることを意識。ダンクのためのバスケをしている。

ゴール下の攻めのレパートリーを増やしたい。

それと最近妙に3pが入る。プッシュプレスのおかげか。

## ピラティス

時間がなくてあまり行けていない。そのせいか最近腰が痛い。1ヶ月ほど前にフロントスクワットで腰をいわしかけた。クリーンで怪我しそうな予感は今の所ない。

家でもヤムナボールをあんまりできていないし。
